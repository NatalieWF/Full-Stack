# Full-Stack

Este projeto consiste em uma aplicação Full Stack completa voltada para o gerenciamento e cadastro de usuários. O objetivo principal é demonstrar a interação real entre uma interface gráfca moderna e um servidor de alta perfomance conectado a um banco de dados na nuvem.

A aplicação é estruturada em três pilares principais:

- Front-end (Interface): Desenvolvido com React e Vite, o sistema oferece uma interface amigável com  um formulário para entrada de nome, idade e e-mail, além de uma listagem dinâmica que exibe os usuários cadastrados em tempo real.

- Back-end (Servidor): Construído com Node.js e o framework Express, ele funciona como uma API REST que recebe as requisições do navegador, processa as regras de negócio e gerencia a comunicação com o banco de dados através de métodos HTTP como GET, POST, PUT e DELETE.

- Banco de Dados e Persistência: Utiliza o MongoDB Atlas para armazenamento na nuvem e o Prisma (ORM) para mapear e manipular os dados de forma eficiente e segura.
